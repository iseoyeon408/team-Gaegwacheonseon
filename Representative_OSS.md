## 유명 oss 4개 조사
### 1. linux 현대 컴퓨팅의 기반
- 핵심정보:운영체제(os) 커널. 1991년 Linus Torvalds 시작.
- 라이선스: Gpl v2 (GNU general public license version 2)
- 기술적 의의: 모눌리식 커널 : 효율성과 성능 최적화에 중점을 둔 설계.
- 모듈성 :실행 중에 커널 모듈을 로드/언로드 하여 유연성을 확보.
- 표준화 : POSIX 표준을 준수하여 다양한 유닉스 시스템과의 호환성을 제공.
- 산업적 영향: 서버 및 클라우드:AWS,Azure 등 모든 주요 클라우드 서비스의 근간.
- 모바일: android 운영 체제의 커널로 활용되어 전세계 스마트폰의 대부분을 구동.
- 임베드/IoT:가볍고 유연하여 라우터, 스마트 가전 등 다양한 임베디드 장치에 사용.
### 2. Git/Git hub: 개발 협업 표준
- 핵심 정보: 분산 버전 관리 시스템(DVCS)및 호스팅 플랫폼.
#### 라이선스 
- Git v2 /GitHub: 플랫폼 이용 약관+개별 프로젝트 라이선스
- 기술적의의 : 분산형 구조:  중앙 서버 없이도 모든 개발자가 완전한 히스토리를 로컬에 보유하여 오프라인 작업 및 안전성 확보
- Merkle tree: 데이터 무결성 보장하는 강력한 해시 기반구조. Barching&merging: pull request 기반의 비선형적이고 동시적인 개발 문화 확립
#### 산업적 영향
- 개발표준 : 전 세계 소프트웨어 개발 및 유지보수 사실상 표준 도구.
- 오픈소스 생태계 활성화: 코드 공유, 버그보고, 기여활동의 장을 제공하여 오픈소스 프로젝트의 투명성과 접근성을 극대화
### 3. Tensor Flow & PyTorch: AI 혁신의 엔진
- 핵심정보: 딥러닝 모델 개발 및  배포를 위한 오픈소스 라이브러리.
- 라이선스: Apache License 2.0 (ASL2.0) (상업적 사용에 매우 자유로움)
#### 기술적 의의
- 자동 미분: 복잡한 신경망 훈련에 필수적인 기울기 계산을 자동화.
- Tensor: 다차원 배열 데이터 구조를 효율적으로 처리.
#### API 추상화
- 고수준 API 추상화: 고수준 API  를 제공하여 모델 구축을 단순화하고  진입 장벽을 낮춤.
- 산업적 영향: 마이크로서비스
- 아키텍처: 복잡한 대규모 애플리케이션의 운영 및 관리를 효율적으로 가능하게 함.
- 멀티/하이브리드 클라우드: 특정 클라우드 벤더에 종속되지 않고 이식성(Portability)을 극대화하여 클라우드 네이티브 환경의 핵심 도구로 자리매김.
